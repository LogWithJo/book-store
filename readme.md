# 📚 Book Store Mini Project

## ✨ Description

The **Book Store Mini Project** is a simple and interactive web application built using **HTML, CSS, Tailwind CSS, and JavaScript**. It allows users to browse, view, and manage books in a clean and responsive UI.

This project is designed to practice:

- DOM manipulation with JavaScript ⚙️
- Responsive UI design using Tailwind CSS 🎨
- Fetching and handling dynamic data 📡
- Building real-world frontend project structure 🧠

It simulates a basic online bookstore where users can explore books, view details, and interact with the interface smoothly across devices.

---

## 🚀 Features

- 📖 Display list of books dynamically
- 🔍 Search/filter books easily
- 🧾 View book details (title, author, price, image)
- ➕ Add new books (if implemented)
- 💾 Local storage support (if used in your version)
- 📱 Fully responsive design (mobile + desktop)
